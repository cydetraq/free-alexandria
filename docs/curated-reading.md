# Canonical linked-only reading list

This file contains only the linked-only works in the Free Alexandria canonical collection. Membership is controlled by `catalog/canonical-lock.json`, which is pinned to the synced Athenaeum PKB record. These works are not supplied as local EPUB/PDF files because the repository does not currently have redistribution rights.

Every entry has a stable ID and an individually curated exact destination. Generated search links are not permitted as primary destinations.

## Validation rules

- The table must contain exactly the IDs in `linked_only` within `catalog/canonical-lock.json`.
- Every ID must appear exactly once.
- Every destination must be an exact author, publisher, institutional, or library record.
- Generic Libby, Google, Amazon, Open Library, WorldCat, or other generated search URLs are prohibited as primary destinations.
- Candidate works not yet approved for canonical membership belong in the PKB research queue, not in this reader-facing list.
